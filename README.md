A minimalistic Tumblr theme for my blogs.

### Notes

- RTL Support: Tag posts as `#hebrew` or `#rtl`.

- Code Highlighting: Done automatically by [HighlightJS](https://highlightjs.org/).

- Formatting poetry: Add `<br>`s like a pleb.
